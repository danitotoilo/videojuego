Juego de Programación en C++ - Daniel Barreto

Instrucciones: En cada nivel debes resolver un ejercicio de programación en C++ (lenguaje C). Tienes 3 pistas disponibles por nivel y 5 intentos antes de perder. Recuerda que debes tener en cuenta los siguientes conceptos para los códigos: - Float: es para definir números con decimales. - Int: es para definir números enteros. - Siempre tienes que empezar con #include "stdio.h", main(). - Los formatos para relacionar el tipo de dato son los siguientes: float=%f, int=%d. - Se utiliza printf para mensajes o para mostrar un resultado: printf("mensaje %formato", variable);. - Se utiliza scanf para capturar datos digitados por el usuario: scanf("formato", & variable);. ¡Buena suerte!
